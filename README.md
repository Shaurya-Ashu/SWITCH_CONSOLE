🎮 Switch Console (ESP32 Gaming Device)
A DIY handheld gaming console built using ESP32, OLED display, and buttons.
🚀 Features
🎮 Multiple games (Tic-Tac-Toe, Snake, Mario)
🖥️ OLED display support
🔘 Button-based controls
💾 High score saving (EEPROM)
🔋 Portable design
🧰 Hardware Used
ESP32 (XIAO ESP32-S3 recommended)
OLED Display (SSD1306)
Push Buttons
Battery (optional)
🔌 Circuit Diagram
See /hardware/PCB/ for full design files.
💻 Firmware
Located in /firmware/
Upload using Arduino IDE.
⚙️ Installation
Install Arduino IDE
Add ESP32 board
Open .ino file
Upload to ESP32
