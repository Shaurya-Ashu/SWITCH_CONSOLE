## 🚀 Features

- 🎮 Multiple Games (Tic-Tac-Toe, Snake, Mario)
- 🖥️ OLED Display (SSD1306)
- 🔘 Button Controls (Up / Down / Select)
- 💾 High Score Saving (EEPROM)
- 🔋 Portable & Compact Design
- ⚡ Fast performance using ESP32-S3

---

## 🧠 About the Project

This project is a **custom-built gaming console** designed for learning embedded systems, electronics, and game development.

It was created as part of a hardware learning initiative and can be extended with:
- Bluetooth game uploads 📡  
- Mobile controller 📱  
- Sound effects 🔊  

---

## 🧰 Hardware Components

| Component | Description |
|----------|------------|
| ESP32 (XIAO ESP32-S3) | Main microcontroller |
| OLED Display | 128x64 SSD1306 |
| Push Buttons | User input |
| Battery (optional) | Portable power |
| Custom PCB | Designed in EasyEDA |

### 🔧 How to Upload

1. Install Arduino IDE  
2. Install ESP32 board package  
3. Open `.ino` file  
4. Select board: **ESP32 / XIAO ESP32-S3**  
5. Click Upload
